# 🤖 Portfolio Telegram Bot (aiogram 3.x)

Интерактивный бот-портфолио с интеграцией ИИ, разработанный для демонстрации навыков backend-разработки.

## 🚀 Технологии
* **Язык:** Python 3.12
* **Фреймворк:** aiogram 3.x (Long Polling)
* **ИИ Модель:** Gemini 2.0 Flash (через Google Generative AI SDK)
* **База данных:** SQLAlchemy + aiosqlite (SQLite)
* **Конфигурация:** Pydantic Settings (Environment variables)
* **Деплой:** Render

## 🛠 Функционал
- **AI-собеседник:** Интеграция с Gemini для ответов на вопросы о моем опыте.
- **Инфо-карточки:** Быстрый доступ к проектам, GitHub и социальным сетям.
- **Админ-панель:** Просмотр статистики пользователей (в разработке).
- **Валидация данных:** Строгая типизация через Pydantic.

## ⚙️ Локальный запуск
1. Клонируйте репозиторий.
2. Создайте виртуальное окружение: `python -m venv venv`.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Создайте файл `.env` на основе `.env.example`.
5. Запустите бота: `python main.py`.
